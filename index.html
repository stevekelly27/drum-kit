Virtual Expierence of Drum Pads

This is a virtual drum-pad that is activated when a pad is clicked on or if you use the corrisponding keys below. Please scroll down to get full effect of all pads.

Order from first pad on top-left to right:

  • r = Triangle
  • t = Closed-Hihat
  • y = Open Hihat
  • f = Tom
  • g = Clap
  • h = Crash
  • v = Kick
  • b = Snare
  • n = Bomb Blast
